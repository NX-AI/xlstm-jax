xlstm_jax.kernels.mlstm_chunkwise.max_triton_fwbw_v5xlchunksize._parallel_bw_dV.mlstm_chunkwise__parallel_bw_dV(matQ, matK, matV, vecI, vecA, vecB, matC_all, vecN_all, scaM_all, vecN_out, vecM_out, matDeltaH, matDeltaC_states, qk_scale=None, chunk_size=64, siz_b_LQ=32, siz_b_LKV=32, siz_b_DHQK=None, siz_b_DHHV=None, num_warps=None, num_stages=None, eps=0.0, output_dtype=jnp.float32)

Computes only the deltaK gradients for the backward pass. The other gradients are computed in the other (kernel) function.

This function defines the grid and block sizes for the kernel launch and calls the kernel. chunk parallel size: siz_b_LKV chunk loop size: siz_b_LQ head dim parallel size: siz_b_DHHV head dim loop size: siz_b_DHQK

Parameters:

  • matQ (jax.Array) – Tensor containing the query vectors. Shape (B, NH, S, DHQK).

  • matK (jax.Array) – Tensor containing the key vectors. Shape (B, NH, S, DHQK).

  • matV (jax.Array) – Tensor containing the value vectors. Shape (B, NH, S, DHHV).

  • vecI (jax.Array) – Tensor containing the input gate pre-activations. Shape (B, NH, NC, L).

  • vecA (jax.Array) – Tensor containing the summed input and cumulative forget gate pre-activations. Shape (B, NH, NC, L).

  • vecB (jax.Array) – Tensor containing the cumulative forget gate pre-activations. Shape (B, NH, NC, L).

  • matC_all (jax.Array) – Tensor containing the C states at the chunk borders. Shape (B, NH, (NC+1) * DHQK, DHHV).

  • vecN_all (jax.Array) – Tensor containing the N states at the chunk borders. Shape (B, NH, (NC+1) * DHQK).

  • scaM_all (jax.Array) – Tensor containing the M states at the chunk borders. Shape (B, NH, (NC+1)).

  • vecN_out (jax.Array) – Tensor containing the normalizer output N. Shape (B, NH, S).

  • vecM_out (jax.Array) – Tensor containing the max state M. Shape (B, NH, S).

  • matDeltaH (jax.Array) – Tensor containing the incoming H gradients. Shape (B, NH, S, DHHV).

  • matDeltaC_states (jax.Array) – Tensor containing the incoming C gradients. Shape (B, NH, (NC+1) * DHQK, DHHV).

  • qk_scale (float | None) – Scaling factor for the QK matrix. Defaults to None.

  • chunk_size (int) – Chunk size. Defaults to 64.

  • siz_b_LQ (int) – Block size for the chunk dimension LQ. Defaults to 32.

  • siz_b_LKV (int) – Block size for the chunk dimension LKV. Defaults to 32.

  • siz_b_DHQK (int | None) – Block size for the head dimension DHQK. Defaults to None.

  • siz_b_DHHV (int | None) – Block size for the head dimension DHHV. Defaults to None.

  • num_warps (int | None) – Number of warps. Defaults to None.

  • num_stages (int | None) – Number of stages. Defaults to None.

  • eps (float) – Epsilon value. Defaults to 1e-6.

  • output_dtype (jax.numpy.dtype) – Output data type. Defaults to jnp.float32.

Returns:

Tensor containing the K gradients. Shape (B, NH, S, DHQK).

Return type:

jax.Array
